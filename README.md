# Music Player 🎵

> A browser-based music player with full playback controls, track navigation, and volume management — built in pure vanilla JavaScript.

🔗 **[Live Demo](https://music-player-two-navy.vercel.app/)**

## ✨ Features

- ▶️ **Play / Pause** — instant toggle with a single click
- ⏭️ **Skip Tracks** — forward and backward navigation
- 🔊 **Volume Control** — smooth slider for audio adjustment

- DOM manipulation for dynamic track info updates

## 🚀 Run locally
```bash
# Clone the repo
git clone https://github.com/FrontendDevSQ/music-player.git

# Open in browser
cd music-player-ui
open index.html
```
No installs. No build step. Just open and play.
